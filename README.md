## Getting Started

Clone the repository:

```bash
git clone https://github.com/pranaygp/workflow-agent-companion-docs.git
cd workflow-agent-companion-docs
```

Install dependencies:

```bash
npm install
# or
pnpm install
# or
yarn install
```

Then start the development server:

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```

Open http://localhost:3000 in your browser to view the detailed workshop instructions.
